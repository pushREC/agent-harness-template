# Web App Builder Example

Example specification for building a task manager web application.

## Usage

```bash
# Copy app_spec.txt to your project prompts folder
cp app_spec.txt /path/to/project/prompts/app_spec.txt

# Run the agent
python autonomous_agent.py --project-dir ./task-manager
```

## What Gets Built

- React frontend with Tailwind CSS
- Node.js/Express backend
- SQLite database
- Full CRUD for tasks
- Search and filtering
- Responsive design
