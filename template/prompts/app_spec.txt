## Example: Task Manager App

Replace this section with your actual specification.

### Overview
A simple task management application for personal productivity. Users can
create tasks with titles, descriptions, due dates, and priorities.

### Technology Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: SQLite

### Core Features

1. **Task Creation**
   - User clicks "Add Task" button
   - Modal appears with form fields
   - User enters title (required), description, due date, priority
   - User clicks "Save" to create task
   - Task appears in the list

2. **Task List Display**
   - All tasks displayed in a responsive grid/list
   - Shows title, due date, priority badge
   - Overdue tasks highlighted in red

3. **Task Completion**
   - Checkbox next to each task
   - Click checkbox to mark complete
   - Completed tasks show strikethrough
   - Can filter to hide completed

4. **Task Deletion**
   - Delete button on each task
   - Confirmation dialog appears
   - Task removed after confirmation

### UI Requirements
- Clean, minimal design
- Blue primary color (#3B82F6)
- White background, gray secondary elements
- Mobile-responsive layout
